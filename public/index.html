Basic Playlist Endpoints

method endpoint usage returns
POST /api/playlists Create new playlist Playlist
GET /api/playlists Get list of playlists Playlist[]
GET /api/playlists/{playlistId} Get playlist by id Playlist
PUT /api/playlists/{playlistId} Update playlist by id Playlist
DELETE /api/playlists/{playlistId} Delete playlist by id Playlist
